# BIC‑ISMRM 2026 Meeting Registration

This repository contains information for managing the BIC‑ISMRM 2026 scientific meeting webpage, including registration and abstract submission details.

## Overview

The webpage is updated in close coordination with the meeting’s Local Organising Committee (LOC) chair. Content includes:

- About the Meeting  
- Pre-Meeting Workshops  
- Registration Options  
- Abstract Submission  
- Contact Information  
- Additional Information  
- Sponsors  

## Navigation

To ensure clarity and avoid confusion:

- The landing page must not contain direct links to external registration or abstract submission systems.
- All users must be directed via the [Registration & Abstract Submission Details page](https://bic.ismrm.org/26m/), which acts as the single entry point to all registration and submission information.

## Roles and Responsibilities

- **Web Manager**: implements updates and manages publishing  
- **LOC Chair**: provides source content and reviews/approves webpage content  

## Content Publishing Workflow

1. Build content in WordPress (draft/preview mode)  
2. Share preview with the LOC Chair for review  
3. Address feedback and revise content  
4. Publish updates to the live site  

## Content Management Guidelines

### Registration and Forms

- The landing page must not contain direct links to registration or abstract submission portals  
- All links must be accessed via the dedicated [Registration & Abstract Submission Details page](https://bic.ismrm.org/26m/)  
- Pre-meeting workshops and the main conference may use different registration systems  
- Clearly signpost users to external systems (e.g. registration, abstract submission, workshops)  
- Ensure consistent link formats (use text hyperlinks rather than QR codes unless justified)

### Visibility and Accessibility

Key information and links must be:

- Easy to find without extensive scrolling  
- Clearly distinguishable from surrounding text  
- Not embedded within long paragraphs  

Ensure the following are prominently displayed:

- Abstract submission deadline
- Links to workshop registration  

Use **bold formatting** and increased font size where appropriate to highlight key links.

### Placeholders

When external links are not yet available, use **“opening soon”** placeholders. This may include:

- Abstract submission portal  
- Registration portal for pre-meeting workshops  
- Workshop registration links (e.g. King’s eStore)  

Replace placeholders as soon as links become available.

### Sponsors

- Organise sponsors by tier (Gold, Silver, Bronze)
- Ensure logos have consistent visual size (adjust as needed)  
- Maintain alignment and spacing across all logos
